    Etiquetas Semanticas Extructurales

    div -Es una etiqueta contenedora que no tiene semantica
    header -Cabecera de un sitio web o de una seccion
    main -Define la seccion principal del documento, solo puede exixtir una etiqueta main por documento
    footer -Pie de la paguina de un sitio web o de una seccion
    aticle -Representa una seccion autocontenido(que por si sola se explica)
    aside -Representa contenido complementario o secundario
    seccion -Representa una seccion de contenido generico
    address -Representa una informacion de contacto
    ☝️

    Etiquetas de Bloques VS Etiquetas de Linea

    La etiqueta de Bloque por exelencia es la div, una etiqueta de bloque es aquella que genera saltos de linea con sus etiquetas hermanas
    La etiqueta de linea por exelencia es la div, una etiqueta de linea solo ocupa el espacio necesarioque tiene su contenido Hola soy otra span ☝️
